# TestPixelPulse
Тестовый проект на Ruby для тестирования основного проекта.
За основу взяты старые макеты *.html оригинального проекта PixelPulse для изучения функционала тестирования и создание проекта на Ruby on Rails.
На данный момент имеется:
- Страницы: `Главная`, `Новости`, `Авторизация`, Регистрация;
- Динамичные страницы. Каждая страница собирается по кускам из `Шапки`, `Содержимого` и `Подвала`. Страница `Новости` собирается по кускам из здешней базе данных;
- Сущность `news`, содержит перечень новостей для тестирования динамичности сайта;
- Автотесты, чек-лист и тест-кейсы.
